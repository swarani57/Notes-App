#  Notes App

A simple and clean Notes Management Web Application built using **HTML, CSS, and JavaScript**.  
Users can create and delete notes, and all notes are **stored in browser localStorage**, so they remain even after refreshing the page.

---

 Features

-  Add notes with title & description  
   Display notes in a beautiful grid layout  
-  Random pastel background for each note  
-  Delete note anytime  
-  localStorage persistence (notes stay after refresh)  
-  Responsive UI Design  
-  Fast & lightweight (no frameworks required)  

---
